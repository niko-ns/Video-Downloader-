# Video Downloader

Простая и удобная утилита для скачивания видео с популярных веб-сайтов. Этот инструмент позволяет сохранять видео прямо на ваше устройство, поддерживает выбор качества и формата.

## Возможности

- ⬇️ Скачивание видео с YouTube, Vimeo, Dailymotion и других платформ.
- 🎚️ Выбор качества видео (1080p, 720p, 480p и т.д.).
- 🎵 Извлечение аудио из видео (конвертация в MP3).
- 📁 Простой интерфейс командной строки.
- ⚡ Поддержка пакетного скачивания (по списку ссылок).
